#  Fullstack Learning Lab

Bienvenido a mi laboratorio de aprendizaje fullstack. Este repositorio está diseñado para practicar **React, Angular y Java Spring Boot**, con proyectos de nivel **fácil → intermedio → avanzado**, incluyendo conceptos y teorías claves en cada etapa.

Te recomiendo empezar por **React** (fácil y visual, resultados rápidos), luego **Java + Spring Boot** (para aprender backend y APIs), y finalmente **Angular** (más complejo, ideal para consolidar todo y proyectos grandes).

## React

###  Fácil
**Proyectos básicos:**
1. **To-Do List**
   - Conceptos: Componentes, Props, State, Eventos.
   - Aprendizaje: Crear, eliminar y marcar tareas completadas.
2. **Calculadora**
   - Conceptos: JSX, manejo de eventos, funciones simples.
   - Aprendizaje: Operaciones básicas, manejo de input y display.
3. **Reloj / Temporizador**
   - Conceptos: useState, useEffect.
   - Aprendizaje: Actualización periódica del DOM.

###  Intermedio
1. **Buscador de películas**
   - Conceptos: Fetch API, promesas, renderizado condicional.
   - Aprendizaje: Consumir API pública (OMDB), mostrar resultados dinámicos.
2. **Galería de imágenes**
   - Conceptos: Componentes dinámicos, props avanzadas.
   - Aprendizaje: Paginación, filtros y lightbox.
3. **Formulario con validaciones**
   - Conceptos: Validaciones, manejo de formularios controlados.
   - Aprendizaje: Feedback en tiempo real, manejo de errores.

###  Difícil
1. **Dashboard tipo Netflix**
   - Conceptos: Rutas con React Router, estado global, hooks avanzados.
   - Aprendizaje: Múltiples APIs, filtrado de contenido, layout responsive.
2. **Red social simple**
   - Conceptos: Context API o Redux, consumo de backend.
   - Aprendizaje: Publicaciones, likes, comentarios y autenticación básica.

##  Tips de aprendizaje estilo “Campus 42”
- **Proyectos fáciles primero** → Termínalos completos antes de subir de nivel.  
- **Aprende teoría mientras practicas** → Por ejemplo, Hooks en React, Observables en Angular, JPA en Spring Boot.  
- **Divide y conquista** → Cada proyecto en su carpeta y con README propio.  
- **Documenta tus avances** → Screenshots, errores y soluciones.  
- **Itera y mejora** → Vuelve a proyectos antiguos y aplica lo nuevo que aprendiste.

## Angular

###  Fácil
1. **Lista de tareas**
   - Conceptos: Componentes, binding de datos, eventos.
   - Aprendizaje: Crear, eliminar, marcar completadas tareas.
2. **Contador simple**
   - Conceptos: Property binding, Event binding.
   - Aprendizaje: Incrementar/decrementar valores y mostrar en template.
3. **Formulario básico**
   - Conceptos: NgModel, validaciones simples.
   - Aprendizaje: Captura de datos y validación de inputs.

###  Intermedio
1. **Buscador de usuarios**
   - Conceptos: Services, HttpClient, observables.
   - Aprendizaje: Consumir API y mostrar resultados dinámicos.
2. **Galería de productos**
   - Conceptos: Componentes hijos, inputs y outputs.
   - Aprendizaje: Filtros y comunicación entre componentes.
3. **Panel administrativo simple**
   - Conceptos: CRUD, rutas, services.
   - Aprendizaje: Gestión básica de datos y navegación.

###  Difícil
1. **Dashboard completo**
   - Conceptos: Routing avanzado, guards, servicios, RxJS.
   - Aprendizaje: Gráficos dinámicos, filtros, layout responsive.
2. **Sistema de gestión de proyectos**
   - Conceptos: Roles de usuario, módulos, Lazy Loading.
   - Aprendizaje: CRUD completo con validaciones y seguridad básica.
